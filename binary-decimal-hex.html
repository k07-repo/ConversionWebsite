Binary, Decimal, Hexadecimal Number Converter

Number can be represented using several different bases. The most common bases used today are decimal (base-10), binary (base-2), and hexadecimal (base-6). A base is the main counting unit. To be more specific, the nth digit from the right represents (base value)^(n - 1). So in base-10, the 3rd digit represents 10^(3 - 1) = 10^2 = 100, so the third digit is the hundreds place.

-Decimal numbers are the ones we use in everyday life.
-Binary numbers use only 0s and 1s and are used in computing applications.
-Hexadecimal numbers use sixteen digits (10-15 are represented by A-F). These are also used in computing applications.
