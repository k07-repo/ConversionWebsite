RGB, Hex, Integer Color Converter

Colors can be represented in multiple different ways, all of which are identical at their core. Provided here are three of the most common.

-RGB: "red", "green", "blue" values; represents the portion of red/green/blue inside the color. Essentially the root of the other color notations.
-Hex: Presented as a six digit hexadecimal (base 16) number. The first two digits represent the RGB value for red, and so on.
-Integer: An ordinary integer that uniquely represents any possible RGB value. Calculated as 65536r + 256g + b.

The provided calculator provides easy conversion between the three. Simply entering any type of value (RGB, Hexadecimal, Integer) will cause the others to fill in automatically, and the color panel to update instantaneously to match.
