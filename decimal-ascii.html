Decimal to ASCII Converter

ASCII - short for American Standard Code for Information Interchange - is an encoding standard that represents basic characters. Each character is given a number from 0-128 (with the first several spots being used for technical characters).

ASCII is often built on by other encoding systems such as UTF-8 nowadays, but it is still used in some places.

The following converter will instantaneously bulk convert a spaced decimal number message it into ASCII plaintext, while typing in ASCII plaintext will convert it into decimal in the same way. For simplicity, only ASCII characters and codes 0-127 are allowed, which covers all English plaintext and keyboard symbols.
